# Cyber Threat Intelligence Portfolio – Shiri Barzilay

## About Me
Cyber SIGINT Investigator and Computer Science student building practical capabilities in Cyber Threat Intelligence (CTI).
This repository documents my structured learning process, threat analysis work, MITRE ATT&CK mappings, and intelligence-style reporting exercises.
My focus is on understanding attacker behavior, campaign lifecycles, and translating technical activity into actionable intelligence insights.

## Core Focus Areas
- Threat actor profiling
- Campaign lifecycle analysis
- TTP mapping (MITRE ATT&CK)
- Indicator identification (IOCs)
- Detection & mitigation considerations
- Intelligence reporting methodology

## Repository Structure

### 📁 CTI-Notes
Summaries and structured notes from:
- TryHackMe (CTI-related rooms)
- Threat research
- Security blogs and reports

### 📁 Threat-Reports
Independent analyses of real-world cyber campaigns.
Each report includes:
- Overview
- Actor profile
- Attack lifecycle breakdown
- MITRE ATT&CK mapping
- Indicators of compromise
- Detection considerations

### 📁 MITRE-Mapping
Focused exercises mapping techniques to:
- Tactics
- Real-world use cases
- Defensive implications

### 📁 IOC-Tools
Small Python utilities for parsing and structuring indicators.

## Methodology
My analytical approach includes:
1. Identifying the threat context
2. Understanding attacker intent and targeting
3. Breaking down technical execution
4. Mapping behaviors to MITRE ATT&CK
5. Extracting structured indicators
6. Considering detection and mitigation strategies
